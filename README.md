# sEMG-Feedback

A low-cost surface electromyography feedback device based on <b>Arduino Mega</b> and <b>Advancer Technologies Muscle Sensor v3</b>.

Prototype: https://www.youtube.com/watch?v=2l5GvTqeB1E
